## Overview

디스코드 상에서 사용하고 싶었던 기능들을 봇을 통해 구현했습니다. 구현 과정은 [블로그]('https://velog.io/@blue4683/posts')에서 확인할 수 있습니다.

**현재 구현된 기능들은 다음과 같습니다.**

- 관리자 기능(유저 추가, 기능 관리)
- 게임 기능(베팅)
- 알림 기능(디스코드 라이브 방송 구독 및 알람)
